* locdog

** Usage

write $HOME/.config/locdog/config.json as below

#+begin_src json
   {
      "default_interval": 5,
      "default_threshold": 60,
      "default_alert_cmd": [
          "echo"
      ],
      "targets": [
          {
              "alert_cmd": [
                  "terminal-notifier",
                  "-title",
                  "LocDog",
                  "-sound",
                  "default",
                  "-message",
                  "warning!"
              ],
              "watch_cmd": [
                  "false"
              ]
          }
      ]
  }
#+end_src
