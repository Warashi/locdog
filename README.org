* locdog

** Usage

write $HOME/.config/locdog/config.json as below

#+begin_src json
  {
      "targets": {
          "watch_cmd": [
              "false"
          ],
          "alert_cmd": [
              "terminal-notifier",
              "-title",
              "LocDog",
              "-sound",
              "default",
              "-message",
              "warning!"
          ]
      },
      "default_alert_cmd": [
          "echo"
      ],
      "default_threshold": 60,
      "default_interval": 5
  }
#+end_src
